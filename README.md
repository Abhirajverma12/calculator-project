# 🧮 Advanced Calculator Web App

A fully functional and responsive calculator web application built using HTML, CSS, and JavaScript.

## 🚀 Live Demo
🔗 https://calculator-project-hazel.vercel.app/

## 📌 Features
- Dark / Light mode toggle 🌙☀️
- Responsive design (mobile-friendly)
- Keyboard support ⌨️
- Real-time calculation
- Calculation history display
- Smooth UI with modern design

## 🛠️ Tech Stack
- HTML5
- CSS3
- JavaScript (Vanilla JS)
